## 🏛️ About the Project

Shrisha Education is an **officially authorized MKCL & MS-CIT training center** in Bhatkuli, Amravati, Maharashtra. They needed a professional website to:

- Showcase their government-recognized courses
- Allow students to enquire & apply online
- Build trust and digital presence in their region

This is a **real-world client project** — live, functional, and actively used.

> 🌐 Live at: [meherp5.netlify.app](https://meherp5.netlify.app)

---

## ✨ Features

| Feature | Description |
|---|---|
| 📋 **Online Enquiry Form** | Students can enquire about any course — team gets notified within 24 hours |
| 🎓 **Course Listings** | All 15+ MKCL & KLiC courses listed with details |
| 📱 **Fully Responsive** | Works perfectly on mobile, tablet and desktop |
| 🖼️ **Gallery Section** | Showcases the center's labs, classrooms and student life |
| ⭐ **Student Reviews** | Real testimonials from MS-CIT & other graduates |
| 📍 **Contact & Location** | Address, phone, WhatsApp link and timings |
| 💬 **WhatsApp Integration** | Direct WhatsApp chat button for instant enquiries |
| 🎯 **Admission Process** | Clear 4-step process explained for new students |

---

## 🛠️ Tech Stack

```
HTML5       →  Structure & Semantic Markup
CSS3        →  Animations, Transitions, Responsive Design
JavaScript  →  Form Handling, Interactions, Smooth Scroll
Netlify     →  Deployment & Hosting
```

No frameworks. No libraries. **Pure vanilla web tech.**

```

## 🚀 Getting Started

**Option 1 — Visit Live Site**
```
👉 https://meherp5.netlify.app
```

**Option 2 — Run Locally**
```bash
# Clone the repo
git clone https://github.com/mehergautam/shrisha-education.git

# Open in browser
cd shrisha-education
open index.html
```

No setup. No install. Just open and run. 🎉
